## 수정 내용

`renderStateJSX()`에서 `.loading`/`.error` condition 처리 시, field 이름 대신 부모 fetch의 `dataVar`를 기반으로 변수명 생성.

```go
// 수정 전
case strings.HasSuffix(s.Condition, ".loading"):
    field := strings.TrimSuffix(s.Condition, ".loading")
    cond = toLowerFirst(field) + "Loading"   // → "coursesLoading" (미정의)

// 수정 후
case strings.HasSuffix(s.Condition, ".loading"):
    cond = dataVar + "Loading"               // → "listCoursesDataLoading" (useQuery 정의와 일치)
```

`dataVar`는 `renderChildNodes` → `renderStateJSX` 호출 체인에서 부모 fetch의 alias(`toLowerFirst(operationId) + "Data"`)가 전달된다.
