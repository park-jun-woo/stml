# 수정지시서 001 실행결과

## 항목별 처리

| # | 항목 | 처리 | 비고 |
|---|---|---|---|
| 1 | API import 경로 | `GenerateOptions.APIImportPath` 추가 | 기본값 `@/lib/api`, fullend가 `../api` 등 지정 |
| 2 | react-query 의존성 | `GenerateResult.Dependencies` 반환 | 사용된 import 기반 자동 수집 |
| 3 | API 함수명 케이싱 | STML 변경 없음 | fullend 측 해결 |
| 4 | 출력 폴더 구조 | STML 변경 없음 | fullend가 outDir 조정 |
| 5 | 'use client' | `GenerateOptions.UseClient` 추가 | 기본값 true |

## 변경 파일

- `generator/generator.go` — GenerateOptions, GenerateResult, Generate/GeneratePage 시그니처 변경
- `generator/imports.go` — APIImportPath, UseClient 옵션 적용
- `cmd/stml/main.go` — 새 반환 타입 처리
- `generator/generator_test.go` — 테스트 3개 추가

## fullend 호출 예시

```go
result, err := generator.Generate(pages, specsDir, outDir, generator.GenerateOptions{
    APIImportPath: "../api",
    UseClient:     false,
})
// result.Dependencies: {"@tanstack/react-query": "^5", "react-hook-form": "^7", "react-router-dom": "^6"}
```
